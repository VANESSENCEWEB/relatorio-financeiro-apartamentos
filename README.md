## 🎯 O Problema que Resolvo

Este projeto nasceu da necessidade real de **automatizar a gestão financeira** de apartamentos de temporada, eliminando horas de trabalho manual e proporcionando análises precisas.

### 💔 Dores Identificadas
- ❌ **Relatórios manuais** consumiam 4+ horas mensais
- ❌ **Erros de cálculo** em planilhas complexas  
- ❌ **Falta de padronização** entre relatórios
- ❌ **Análises superficiais** sem insights estratégicos
- ❌ **Dados espalhados** em múltiplas fontes

### ✨ Solução Desenvolvida
- ✅ **Automação completa** - de Excel para PDF em 30 segundos
- ✅ **Análises avançadas** - taxa de ocupação, margem líquida, projeções
- ✅ **Design profissional** - relatórios dignos de apresentação
- ✅ **Zero erros** - cálculos precisos e validados
- ✅ **Insights acionáveis** - recomendações automáticas

## 🚀 Instalação e Uso

### ⚡ Instalação Rápida
```bash
# 1. Clone o repositório
git clone https://github.com/vanessarafaella/relatorio-financeiro-apartamentos.git
cd relatorio-financeiro-apartamentos

# 2. Instale as dependências
pip install -r requirements.txt

# 3. Configure seus dados
cp src/config.py src/config_local.py
# Edite config_local.py com suas informações reais

# 4. Execute para julho 2025
python src/gerador_relatorios.py --mes 7 --ano 2025
```
